# 力扣经典算法实现

## 简介

本项目包含力扣（LeetCode）经典算法题目的实现，目前提供 **Java** 和 **JavaScript** 两种语言版本。

## 目录结构

```
algorithm/
├── Java/          # Java 实现
│   ├── 两数之和P1/
│   ├── 反转链表/
│   ├── 爬楼梯P70/
│   └── ...更多题目
└── JS/            # JavaScript 实现
    ├── 1.js
    ├── 2.js
    └── ...更多题目
```

## 使用方法

```bash
# Java 编译运行
javac -d out src/main/*.java
java -cp out Solution

# JavaScript 运行
node xxx.js
```
